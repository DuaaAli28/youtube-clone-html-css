# YouTube Clone (HTML & CSS)

This project is a static clone of YouTube's homepage built using only HTML and CSS.

## Features:
- Responsive layout
- Header with search bar
- Sidebar navigation
- Video grid layout

## Technologies Used:
- HTML5
- CSS3 (Flexbox, Grid)

## Purpose:
This project was built to practice layout design, responsiveness, and frontend structure.
